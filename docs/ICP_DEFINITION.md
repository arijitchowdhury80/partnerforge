## Composite Scoring

### 4-Factor Model (25% Each)

**1. Fit Score (25%)**
- Vertical match (Fashion/Grocery = 100)
- Company size (Enterprise = 100)
- Geography (US/UK/EMEA = 100)

**2. Intent Score (25%)**
- Hiring signals (engineering roles)
- Tech stack changes
- Traffic growth trends

**3. Value Score (25%)**
- Revenue potential
- Traffic volume
- Deal size estimate

**4. Displacement Score (25%)**
- Uses competitor search (Elasticsearch = 100)
- Uses partner tech (Adobe AEM + Shopify = 100)
- Current search is weak

### Score Ranges

| Total Score | Priority | Action |
|-------------|----------|--------|
| **80-100** | HOT | Ready for outreach |
| **60-79** | WARM | Nurture campaign |
| **40-59** | COOL | Low-touch monitoring |
| **0-39** | COLD | Deprioritize |

## Cohort Definitions

### JACKPOT Cohort (Score: 100)

**Criteria:**
- CMS + Commerce + Hyperscaler (3 partner tech categories)
- OR: CMS + Commerce + Marketing automation

**Why JACKPOT:**
- 3x validation (multiple partner relationships)
- Enterprise-grade investment
- Complex tech stack = high search needs

**Example:** Company using Adobe AEM + Shopify Plus + AWS

### HIGH Cohort (Score: 85)

**Criteria:** CMS + Commerce (2 categories)

**Example:** Adobe AEM + Magento

### STANDARD Cohort (Score: 60-70)

**Criteria:** Commerce OR CMS only

**Example:** Shopify Plus only

## ICP Derivation Methodology

### Data Sources

1. **Customer Logos:** 1,306 analyzed
2. **Customer Quotes:** 379 testimonials
3. **Customer Stories:** 82 case studies
4. **Proof Points:** 81 quantified results

### Validation Process

1. Extract all customer mentions from algolia.com
2. Categorize by vertical, size, geography
3. Identify proof points (quantified metrics)
4. Weight by evidence strength
5. Validate with sales team

### Key Insight

**Traditional ICP said:** "E-commerce is Tier 1"

**Data-driven ICP says:** "Fashion and Grocery are Tier 1 (100% of proof points). General E-commerce is Tier 2 (many customers, no proof yet)."
