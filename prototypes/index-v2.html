Architecture Modules Roadmap
🏗️ Architecture Overview

Key Decision: Production will call APIs directly, NOT through MCP servers

Direct API Approach:
  • BuiltWith: api.builtwith.com REST API
  • SimilarWeb: api.similarweb.com REST API
  • Yahoo Finance: query2.finance.yahoo.com public API
  • Browser: Playwright library directly (no MCP)
  • WebSearch: Perplexity API or Google Search API

MCP servers are for CLI/local dev only. Production needs better caching, monitoring, and rate limiting.

For detailed architecture diagrams, database schemas, and API endpoints:

See architecture-diagram.md in the Dashboard folder.
